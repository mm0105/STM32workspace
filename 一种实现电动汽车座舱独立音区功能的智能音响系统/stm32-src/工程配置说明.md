# STM32F103C8 固件库工程配置说明

## 📁 工程结构

```
work/
├── start/              # 启动文件和核心文件
│   ├── core_cm3.c/h           # Cortex-M3 内核文件
│   ├── startup_stm32f10x_md.s # 启动文件（中容量芯片）
│   ├── stm32f10x.h            # 寄存器定义头文件
│   └── system_stm32f10x.c/h   # 系统时钟初始化
├── library/            # STM32 标准外设库 V3.5
│   ├── stm32f10x_xxx.c/h  # 各外设驱动源文件
│   └── misc.c/h           # NVIC 和 SysTick 配置
├── user/               # 用户代码目录
│   ├── main.c               # 主程序入口
│   ├── stm32f10x_conf.h     # 库配置文件
│   └── stm32f10x_it.c/h     # 中断服务函数
├── Objects/            # 编译输出（.o, .axf 等）
├── Listings/           # 列表文件（.lst, .map 等）
├── DebugConfig/        # 调试配置
├── RTE/                # ARM RTE 组件
├── test_pro.uvprojx    # Keil 工程文件
├── test_pro.uvoptx     # 工程选项
└── keilkill.bat        # 清理脚本
```

## ⚙️ Keil 工程配置

### 1. 芯片选型
- **MCU**: STM32F103C8
- **系列**: STM32F10x Medium-density (中容量)
- **Flash**: 64KB
- **RAM**: 20KB
- **封装**: LQFP48/LQFP64

### 2. 编译器设置
- **编译器**: ARMCC V5.06 update 7
- **C 语言标准**: C99 模式
- **优化等级**: Level 2 -O2

### 3. 头文件包含路径
```
.\start
.\library
.\user
```

### 4. 预定义宏
```
USE_STDPERIPH_DRIVER
STM32F10X_MD
```

### 5. 启动文件选择
根据芯片容量选择对应的启动文件：
- `startup_stm32f10x_ld.s`   - 小容量 (LD): Flash ≤ 32KB
- `startup_stm32f10x_md.s`   - 中容量 (MD): 64KB ≤ Flash ≤ 128KB ✅ **当前使用**
- `startup_stm32f10x_hd.s`   - 大容量 (HD): Flash ≥ 256KB
- `startup_stm32f10x_xl.s`   - 超大容量 (XL): Flash ≥ 512KB

### 6. 时钟配置
- **外部晶振**: HSE = 8MHz
- **系统时钟**: SYSCLK = 72MHz
- **AHB 时钟**: HCLK = 72MHz
- **APB1 时钟**: PCLK1 = 36MHz
- **APB2 时钟**: PCLK2 = 72MHz

## 📝 库文件配置

### stm32f10x_conf.h 配置

此文件用于启用/禁用外设驱动头文件。只启用需要的外设可以减小编译体积。

```c
// 启用所有外设驱动（默认）
#include "stm32f10x_adc.h"
#include "stm32f10x_gpio.h"
#include "stm32f10x_rcc.h"
#include "stm32f10x_usart.h"
// ... 其他外设

// 如果只需要部分外设，注释掉不需要的
// #include "stm32f10x_can.h"
// #include "stm32f10x_cec.h"
```

## 🔧 常用外设初始化示例

### GPIO 初始化（main.c 中已包含）

```c
// 使能 GPIOC 时钟
RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOC, ENABLE);

// 配置 GPIO
GPIO_InitTypeDef GPIO_InitStruct;
GPIO_InitStruct.GPIO_Pin = GPIO_Pin_13;
GPIO_InitStruct.GPIO_Mode = GPIO_Mode_Out_PP;  // 推挽输出
GPIO_InitStruct.GPIO_Speed = GPIO_Speed_50MHz;
GPIO_Init(GPIOC, &GPIO_InitStruct);

// 输出控制
GPIO_SetBits(GPIOC, GPIO_Pin_13);    // 高电平
GPIO_ResetBits(GPIOC, GPIO_Pin_13);  // 低电平
```

### USART 串口初始化

```c
// 1. 使能时钟
RCC_APB2PeriphClockCmd(RCC_APB2Periph_USART1 | RCC_APB2Periph_GPIOA, ENABLE);

// 2. 配置 GPIO
GPIO_InitTypeDef GPIO_InitStructure;
GPIO_InitStructure.GPIO_Pin = GPIO_Pin_9;  // TX
GPIO_InitStructure.GPIO_Mode = GPIO_Mode_AF_PP;
GPIO_InitStructure.GPIO_Speed = GPIO_Speed_50MHz;
GPIO_Init(GPIOA, &GPIO_InitStructure);

GPIO_InitStructure.GPIO_Pin = GPIO_Pin_10;  // RX
GPIO_InitStructure.GPIO_Mode = GPIO_Mode_IN_FLOATING;
GPIO_Init(GPIOA, &GPIO_InitStructure);

// 3. 配置 USART
USART_InitTypeDef USART_InitStructure;
USART_InitStructure.USART_BaudRate = 9600;
USART_InitStructure.USART_WordLength = USART_WordLength_8b;
USART_InitStructure.USART_StopBits = USART_StopBits_1;
USART_InitStructure.USART_Parity = USART_Parity_No;
USART_InitStructure.USART_HardwareFlowControl = USART_HardwareFlowControl_None;
USART_InitStructure.USART_Mode = USART_Mode_Rx | USART_Mode_Tx;
USART_Init(USART1, &USART_InitStructure);

// 4. 使能 USART
USART_Cmd(USART1, ENABLE);
```

### 中断配置

```c
// 配置 NVIC
NVIC_InitTypeDef NVIC_InitStructure;
NVIC_InitStructure.NVIC_IRQChannel = USART1_IRQn;
NVIC_InitStructure.NVIC_IRQChannelPreemptionPriority = 1;
NVIC_InitStructure.NVIC_IRQChannelSubPriority = 1;
NVIC_InitStructure.NVIC_IRQChannelCmd = ENABLE;
NVIC_Init(&NVIC_InitStructure);

// 使能中断
USART_ITConfig(USART1, USART_IT_RXNE, ENABLE);
```

## 🔍 常见问题解决

### 1. 编译错误：找不到头文件
**解决**: 检查 Options → C/C++ → Include Paths 是否包含所有路径

### 2. 下载失败
**解决**: 
- 检查 BOOT0 是否接低电平
- 检查复位电路
- 重新安装 ST-Link 驱动

### 3. 程序不运行
**解决**:
- 检查时钟配置
- 检查启动文件是否正确
- 检查 Reset 和 Clock 设置

### 4. 堆栈溢出
**解决**: 在启动文件中调整 Stack_Size 和 Heap_Size
