## 阅读顺序

1. 先看各项目目录下的 README，了解单个项目的完整说明。
2. 再看代码目录结构，重点关注 `Core/`、`stm32-src/`、`src/` 这类核心业务实现。
3. 如果是面试展示，优先看项目的总体架构、通信协议、状态机和控制逻辑。
